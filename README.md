# Chess Board Simulator

## Pieces
- King
- Queen
- Knight
- Pawn

## Layout
- Rows 1-8, Columns A-H, where A8 is the top left square and H1 is the bottom right square of the chess board

## Sample Input
- Pawn, G1
- King, D5
- Queen, E4
- Knight, H5

## Run
```
go mod vendor
go run main.go
```

## Test
```
go test ./...
```
